**Duración**: 6 horas

Este módulo profundiza en las tareas diarias principales de un administrador de sistemas.

## Temas Cubiertos

Este módulo se divide en las siguientes secciones detalladas:

### [3.1 Gestión de Usuarios y Grupos](/es/module-3/1-users-groups/)
-   Los "Tres Grandes" archivos: `/etc/passwd`, `/etc/shadow`, `/etc/group`.
-   Creación y modificación de usuarios (`useradd`, `usermod`).

### [3.2 Gestión de Paquetes](/es/module-3/2-packages/)
-   **Ubuntu/Debian**: Uso de `apt`.
-   **Rocky/RHEL**: Uso de `dnf`.

### [3.3 Gestión de Servicios con Systemd](/es/module-3/3-services/)
-   Entendiendo las Unidades.
-   Uso de `systemctl` para iniciar, detener, habilitar y comprobar el estado.

### [3.4 Registros y Monitorización](/es/module-3/4-monitoring/)
-   Visualización de registros con `journalctl` y archivos tradicionales en `/var/log`.
-   Programación de tareas con **cron**.
