## Ejercicio 2.4 — Permisos

**Analiza esta salida de `ls -l`:**

```
-rwxr-x--- 1 deploy www-data 4096 mar  9 10:00 deploy.sh
```

¿Quién puede ejecutar el archivo?

<MultipleChoice>
  <Option>Solo el propietario `deploy` y el usuario `root`.</Option>
  <Option isCorrect>El usuario `deploy` (rwx) y cualquier miembro del grupo `www-data` (r-x). Otros usuarios no tienen ningún permiso.</Option>
  <Option>Todos los usuarios del sistema, ya que tiene bit de ejecución.</Option>
</MultipleChoice>

**¿Qué comando da permisos de ejecución al propietario de un script, sin modificar los demás permisos?**

<MultipleChoice>
  <Option>`chmod 777 script.sh`</Option>
  <Option isCorrect>`chmod u+x script.sh`</Option>
  <Option>`chown +x script.sh`</Option>

## Ejercicio 2.8 — Caso práctico integrador

Tienes que encontrar todos los archivos `.log` mayores de 100 MB en el servidor y comprimirlos en un único archivo:

```bash
# Paso 1: Encuenta los archivos
find /var/log -name "*.log" -size +100M

# Paso 2: Comprime los resultados (ejemplo con xargs)
find /var/log -name "*.log" -size +100M | xargs tar -czvf logs_grandes.tar.gz
```

**¿Qué hace `-size +100M` en el comando `find`?**

<MultipleChoice>
  <Option>Filtra archivos de exactamente 100 MB.</Option>
  <Option isCorrect>Filtra archivos con tamaño **mayor** que 100 MB.</Option>
  <Option>Limita la búsqueda a los últimos 100 minutos.</Option>
</MultipleChoice>
