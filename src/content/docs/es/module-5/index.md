**Duración**: 6 horas

Este módulo asegura que puedas conectar tu servidor Linux al mundo de forma segura.

## Temas Cubiertos

Este módulo se divide en las siguientes secciones detalladas:

### [5.1 Configuración de Red](/es/module-5/1-network-config/)
-   Conceptos: IP, Máscara de red, Puerta de enlace, DNS.
-   Configuración: Netplan (Ubuntu) y NetworkManager (Rocky).

### [5.2 Cortafuegos](/es/module-5/2-firewalls/)
-   Configuración de **UFW** (Ubuntu).
-   Configuración de **firewalld** (Rocky).

### [5.3 Configuración Segura de SSH](/es/module-5/3-ssh-security/)
-   Configuración de la autenticación basada en claves.
-   Endurecimiento de `sshd_config` (Deshabilitar root e inicio de sesión con contraseña).

### [5.4 Marcos de Seguridad](/es/module-5/4-security-basics/)
-   Introducción al Control de Acceso Obligatorio (MAC).
-   SELinux (Enforcing/Permissive) y AppArmor.
