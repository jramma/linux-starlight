**Duración**: 4 horas

Este módulo cubre los fundamentos de Linux, certificaciones oficiales y el proceso de instalación y configuración de un servidor por primera vez.

## Temas Cubiertos

Este módulo se divide en las siguientes secciones detalladas:

### [1.1 Visión General y Certificaciones](/es/module-1/1-overview/)
-   El Ecosistema Linux (Debian vs. RHEL).
-   Certificaciones clave: LFCS y RHCSA.

### [1.2 Instalación de Linux](/es/module-1/2-installation/)
-   Guía paso a paso para instalar **Ubuntu Server**.
-   Guía paso a paso para instalar **Rocky Linux**.

### [1.3 Particionamiento y Proceso de Arranque](/es/module-1/3-partitioning-boot/)
-   Estándar de Jerarquía de Sistemas de Archivos (FHS).
-   Comprender las particiones (MBR vs. GPT) y el proceso de arranque de Linux (GRUB, Systemd).

### [1.4 Configuración Inicial](/es/module-1/4-configuration/)
-   Primeros pasos después del reinicio: Nombre de host, Red y SSH.
