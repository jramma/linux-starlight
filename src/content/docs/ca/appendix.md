## Glossari de Termes

### D
**Distro (Distribució)**
Una versió específica de Linux que inclou el nucli, utilitats de sistema, paquets de programari i eines de configuració. Exemples: Ubuntu, Debian, Rocky Linux i Fedora.

### G
**GUI (Interfície Gràfica d'Usuari)**
Una manera visual d'interactuar amb un ordinador mitjançant finestres, icones i menús, utilitzada per la majoria de sistemes operatius moderns. Els servidors Linux sovint s'executen sense GUI (en mode headless) per estalviar recursos.

### K
**Kernel**
El component central d'un sistema operatiu. Gestiona les operacions de l'ordinador i el maquinari, especialment la memòria i el temps de CPU.

### L
**LVM (Logical Volume Manager)**
Marc de gestió de volums lògics per al nucli de Linux. Permet redimensionar particions de manera flexible.

**LTS (Long Term Support)**
Política de cicle de vida de producte per a programari que ofereix estabilitat durant més temps que les edicions estàndard. Les versions Ubuntu LTS es mantenen durant 5 anys.

### P
**Gestor de Paquets**
Conjunt d'eines de programari que automatitza la instal·lació, actualització, configuració i eliminació de programes. Exemples: `apt` (Debian/Ubuntu), `dnf` (RHEL/Rocky).

**POSIX (Portable Operating System Interface)**
Família d'estàndards especificats per la IEEE Computer Society per mantenir la compatibilitat entre sistemes operatius.

### R
**Repositori (Repo)**
Ubicació d'emmagatzematge per a paquets de programari. Els sistemes Linux utilitzen repositoris per descarregar i actualitzar programari de manera segura.

### S
**Shell**
Programa que exposa els serveis d'un sistema operatiu a un usuari o a altres programes. En general, les shells ofereixen una interfície de línia d'ordres (CLI) o una interfície gràfica (GUI).

**SSH (Secure Shell)**
Protocol de xarxa criptogràfic per operar serveis de xarxa de forma segura sobre una xarxa no fiable. És el mètode estàndard per a l'administració remota de Linux.

### S
**Systemd**
Conjunt de programari que proporciona diversos components de sistema per a Linux. El seu objectiu principal és unificar la configuració i el comportament dels serveis entre distribucions Linux.
